### Especificaciones Técnicas y Funcionales

La versión 1.5.0 introduce una arquitectura orientada a la autonomía del estudiante y la persistencia de datos bajo un enfoque *Offline-First*, garantizando acceso continuo a la información académica.

| Módulo del Sistema | Descripción de la Funcionalidad |
| :--- | :--- |
| **Gestión Académica** | Visualización estructurada de calificaciones y cálculo algorítmico de promedios ponderados en tiempo real. |
| **Motor de Proyección** | Simulador integrado para el cálculo de notas finales y evaluación predictiva de exámenes sustitutorios. |
| **Arquitectura Offline** | Acceso ininterrumpido a los registros académicos e historiales sin dependencia de conexión a la red. |
| **Red Social Estudiantil** | *Campus*: Entorno interactivo con sistema de publicaciones, directorio unificado y notificaciones (Realtime). |
| **Sistema de Gamificación** | Implementación de medallas de reconocimiento y *badges* de verificación basados en la actividad del usuario. |

### Protocolo de Instalación

Para proceder con la instalación del paquete alojado en la sección de *Assets*:

1. **Descarga del Binario:** Seleccione el archivo `app-release.apk` ubicado en la parte inferior de esta página.
2. **Autorización del Sistema:** Al tratarse de un paquete distribuido fuera de la tienda oficial, el sistema operativo Android solicitará permisos para la instalación de aplicaciones desde **Orígenes Desconocidos**. Este es un comportamiento estándar.
3. **Validación de Seguridad:** El paquete de instalación se encuentra firmado digitalmente y respaldado por los protocolos de seguridad de nuestra arquitectura base.
